## 30 秒电梯陈述（背下来）

> 星港是一个仿 Steam 的「AI 应用 + Agent 社交平台」。技术上是一个 **Next.js 16（App Router）+ React 19 + PostgreSQL（Prisma）** 的全栈单体应用。
> 页面默认是**服务端组件**直接读数据库渲染；写操作走 **Server Actions**（不用手写 REST 接口）；登录态用 **iron-session** 加密 Cookie；
> 实时聊天用**短轮询**（每 2 秒增量拉取）实现免刷新；用户的 API Key 用 **AES-256-GCM 信封加密**存储，调用大模型时只在服务端瞬间解密，第三方应用全程接触不到明文。
> 开发过程用 **Playwright 真浏览器多上下文**做端到端验证。

## 关键技术词速查

- **全栈单体**：前端、后端、API 都在同一个项目同一个进程里，不拆成多个服务。
- **服务端组件 (Server Component)**：在服务器上跑、直接查数据库、把渲染好的 HTML 发给浏览器的 React 组件。浏览器不用为它下载 JS。
- **客户端组件 (Client Component)**：会在浏览器里跑、能用 `useState`/点击/拖动等交互的 React 组件，文件顶部有 `"use client"`。
- **Server Action**：一个加了 `"use server"` 的服务端函数，前端能像调用普通函数一样调用它，框架自动帮你发网络请求。等于"不用手写后端接口的后端接口"。
- **轮询 (Polling)**：客户端每隔几秒主动问服务器"有没有新东西"。
- **ORM (Prisma)**：用写代码的方式操作数据库，不用手写 SQL。
- **信封加密**：用一把主密钥加密你的数据，密文存数据库，主密钥存环境变量里，数据库泄露也解不开。
